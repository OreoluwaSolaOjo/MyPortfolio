GalaxybyOSG Mobile Application

This is mobile game application which uses land generation technique.

The objective: Create a mobile game application which users tend to see an increasing high score as they play and dodge obstacles

Users of this game will see an increase in speed as the reach certain high score levels. This game was created using Kivy and it had was implemented based on a lot of maths principles.

This project is complex but efficient. I am currently working on updates that will include harder levels and a better user interface. Watch this space.

Technologies: PYTHON, Kivy
